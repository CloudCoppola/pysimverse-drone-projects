# pysimverse-drone-projects

A structured, project-based learning repository for developing drone control, navigation, and autonomous behaviours using PySimverse.

## Purpose

This repository is designed as a structured learning system to develop practical skills in:

- Drone control and dynamics
- Navigation and path planning
- Perception and decision-making
- Autonomous behaviour design

## Prerequisites

- Python (3.10+ recommended)
- PySimVerse installed:
  ```bash
  pip install pysimverse
  ```
